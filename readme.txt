== Installation ==

=== How to install and activate the plugin ===

Please see our [step by step instructions guide](https://ord.to/wordpress) on how to install and use this plugin.

**From within WordPress**

1. Go to `Plugins > Add New`
1. Search by Author for Ordto
1. Activate the plugin from the Plugins page

**Manually**

1. Upload the `ordering-system` folder to the `/wp-content/plugins/` directory
1. Activate the plugin from your Plugins page

=== How to set up your restaurant account ===

1. Look for `Ord.to` in your WordPress menu
1. Go to configuration tab and follow the link on the banner to create ord.to account (if you do not have one already)
1. Configure your products and website inside ord.to
1. Paste your API key and website link into configuration tab
1. Choose view mode: widget or menu button

== Frequently Asked Questions ==

= Why is your solution free? =

Our business model is relying on the idea of a freemium model. There are no hidden fees, commissions or contracts. You have free 10 products and 10 orders every month. If you want more, you can simply buy a subscription.
An answer to that question.

= Is it required to use ord.to system? =

You need to use ord.to system in the very beginning in order to configure your products and website appearance. After that, you will be able to manage all delivery and ordering processes via Wordpress plugin menu.
